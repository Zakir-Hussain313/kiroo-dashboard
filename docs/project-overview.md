# Kiroo — Project Overview

## What This Is
Kiroo is a solo-founder agency operating system built for a premium web design/development agency.

## Business Focus
- High-converting websites
- Landing pages
- Conversion-focused design
- Lead generation websites

## Target Clients
- International businesses
- Service businesses
- Agencies, consultants, founders
- Businesses that understand ROI

## Purpose
- Find high-quality leads
- Identify businesses with weak websites
- Track outreach
- Manage proposals
- Manage onboarding and fulfillment
- Create repeatable systems

## Founder Context
- Solo operator, BSIT student
- ~4 hours/day available
- Low budget, needs simplicity
- Works daily
